# 🧑‍💼 HR Employee Attrition Analysis & Prediction

## 📌 Project Overview
This project analyzes employee attrition to understand the key factors driving employees to leave an organization and builds a machine learning model to predict attrition risk. The insights are visualized through an interactive dashboard to support HR decision-making.

---

## 🎯 Objectives
- Identify key drivers of employee attrition
- Analyze patterns across departments and job roles
- Build a predictive model for attrition risk
- Create an interactive HR analytics dashboard

---

## 📊 Dataset
- HR Employee Attrition dataset
- Total records: 1470 employees
- Features: 34 attributes (demographics, job role, income, satisfaction, etc.)
- Target variable: Attrition (Yes/No → 1/0)

---

## 🛠️ Tools & Technologies
- **SQL** → Data exploration & aggregation analysis
- **Python (Pandas, Scikit-learn)** → Data preprocessing & machine learning
- **Random Forest Classifier** → Prediction model
- **Logistic Regression** → Baseline model
- **Power BI** → Interactive dashboard visualization

---

## 📈 Key Insights
- Overall attrition rate: **~16%**
- Sales department shows the highest attrition rate
- Employees with **over-time work** are more likely to leave
- Lower **monthly income** is strongly associated with attrition
- Job satisfaction and work-life balance are major influencing factors

---

## 🤖 Machine Learning Model
- Model used: Random Forest Classifier
- Accuracy: **~87%**
- Features include income, job role, overtime, satisfaction, and experience-related variables
- Output: Predicts whether an employee is likely to leave or stay

---

## 📊 Dashboard Highlights (Power BI)
- Executive overview of workforce attrition
- Department-wise and job role-wise attrition analysis
- Risk analysis using income, satisfaction, and work-life balance
- Visual comparison of actual vs predicted attrition

---

## 💼 Business Value
This project helps HR teams:
- Identify employees at high risk of leaving
- Understand key factors behind attrition
- Improve retention strategies
- Make data-driven workforce decisions
